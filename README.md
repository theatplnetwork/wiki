# Wiki

The collective knowledge of the ATPL Network

This project is aimed to collect and provide knowledge about aviation, including, but not limited to the path to a European pilot license.

## How to Contribute

### Requirements

- [git](https://git-scm.com/)
- [uv](https://docs.astral.sh/uv/getting-started/installation/)

### Get started

- create your own fork of this project
- clone your fork using git and configure it to contribute to the upstream repository
- within the project execute `uv sync`
- to start local development run `uv run zensical` and `zensical serve`
- make your changes
- If you wish, you can build the project for static usage with `zensical build`. This is normally not required during development

When submitting pull requests please ensure using [semantic commit messages](https://dev.to/yusufbozaci/guide-to-regular-commit-messages-3agd)
